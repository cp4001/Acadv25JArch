# DuctSize_Design.md

> 원본: `DUCT_MEASURE_1204.xls` (시트 `DUCT MEASURE`, `환산표`) 분석 기반
> 목적: 풍량(CMH) 입력 → 등가원형 직경(De) 산정 → 사각덕트(단변×장변) 결정 프로세스의 프로그램화 설계

---

## 1. 개요

### 1.1 처리 목표
풍량 한 가지를 입력하면 다음 두 가지를 출력한다.
- **원형덕트 등가직경 De** [mm]
- **사각덕트 치수 (단변 b × 장변 a)** [mm]

### 1.2 입력/출력

| 구분 | 항목 | 단위 | 비고 |
|------|------|------|------|
| 입력 | 풍량 Q | CMH (m³/h) | 필수 |
| 입력 | 마찰손실 R | mmAq/m | 기본 0.08 |
| 입력 | 여유율 α | (무차원) | 기본 1.01 |
| 입력 (선택) | 단변 b | mm | 지정 시 단일 결과, 미지정 시 모든 조합 |
| 입력 | 아스펙트비 한계 | (무차원) | 기본 **3.1** |
| 출력 | 등가직경 De | mm | 계산 |
| 출력 | 사각덕트 조합 (b × a) | mm | 모드에 따라 1개 또는 N개 |
| 출력(보조) | 면적, 풍속, 동압, 실제저항, De′ (검증), 아스펙트비 | — | 검토용 |

### 1.3 동작 모드

| 모드 | 단변 입력 | 출력 |
|------|---------|------|
| **A. 단변 지정** | 있음 | 해당 단변에 대한 최소 장변 1개 |
| **B. 모든 조합 탐색** | 없음 | De 만족 & 아스펙트비 ≤ 한계 인 (b, a) 조합 전체 |

## 2. 엑셀 셀 맵핑

### 2.1 `DUCT MEASURE` 시트의 핵심 셀

| 셀 | 의미 | 종류 | 수식 / 값 |
|----|------|------|----------|
| D6 | 풍량 Q [CMH] | 입력 | (Named: `풍량`) |
| D8 | 마찰손실 R [mmAq/m] | 입력 | (Named: `저항`) |
| D9 | 여유율 α | 입력 | (Named: `여유율`) |
| D7 | 직경 표시 [mm] | 참조 | `=L3` (별도 영역의 사각→원형 변환 결과를 표시) |
| D14 | De 단면적 [m²] | 계산 | `=ROUND((PI()*D18^2)/4/1000000,3)` |
| D15 | 풍속 V [m/s] | 계산 | `=1.2732*D6/(3600*D18^2)*1000000` |
| D16 | 동압 Δp [mmAq] | 계산 | `=0.061*(D15^2)` |
| D17 | 실제 저항 R′ [mmAq/m] | 계산 | `=0.00119*D15^1.9*(D18/1000)^-1.22` |
| **D18** | **등가직경 De [mm]** | **계산** | `=ROUND(((3.295*10^-10*풍량^1.9/저항)^0.199*1000)/여유율,0)` |
| E25 | 사각 단변 b [mm] | 입력 | 사용자가 표준치수에서 선택 |
| **F25** | **사각 장변 a [mm]** | **계산** | `=(LOOKUP(ROUNDUP($D$18/10,2),환산표!$A5:$IV7)*10)` |
| D25 | 사각 면적 [m²] | 계산 | `=E25*F25/1000000` |
| D26 | De 면적 [m²] | 참조 | `=D14` |
| F26 | 사각 → 등가원형 (검증) [mm] | 계산 | `=ROUND(1.3*((E25*F25)^0.625/(E25+F25)^0.25),0)` |

### 2.2 셀 위치 정정 사항
사용자가 처음 설명한 셀 위치와 실제 엑셀의 셀 위치가 다음과 같이 다르다.

| 사용자 설명 | 실제 엑셀 | 비고 |
|------------|----------|------|
| “E25에 답변 입력 → E26(장변) 결정” | E25(단변 입력) → **F25**(장변 결정) | 본 설계는 실제 셀 기준으로 진행 |
| “L3 수식으로 D7 직경 계산” | L3은 J3·K3(별도 영역)의 사각→원형 변환식. 풍량→직경의 주 흐름은 **D18(De)** | D7은 단순 참조 |

| 이름 | 참조 |
|------|------|

## 3. 계산 흐름

```
┌──────────────────┐
│ 입력: Q, R, α    │
└────────┬─────────┘
         ▼
┌────────────────────────────────────┐
│ Step 1.  De 계산 (D18)             │
│   De = round((3.295e-10·Q^1.9 / R) │
│              ^0.199 · 1000 / α)    │
└────────┬───────────────────────────┘
         ▼
┌────────────────────────────────────┐
│ Step 2.  보조 검토값 산출          │
│   A, V, Δp, R′                     │
└────────┬───────────────────────────┘
         ▼
   ┌─────┴──────┐
   │ 단변 지정? │
   └─┬────────┬─┘
   Yes        No
     │        │
     ▼        ▼
┌─────────┐  ┌──────────────────────────────────┐
│ Mode A  │  │ Mode B (모든 조합)               │
│ Step 3A │  │ Step 3B                          │
│ a 결정  │  │ for each b in standardSizes:     │
│         │  │   최소 a 찾기 (De_eq ≥ De)       │
│         │  │   if a/b ≤ aspectMax: 추가       │
└────┬────┘  └────────┬─────────────────────────┘
     │                │
     └────────┬───────┘
              ▼
┌────────────────────────────────────┐
│ Step 4.  각 (b,a)에 대한 검증       │
│   De′  = 1.3·((b·a)^0.625 /        │
│              (b+a)^0.25)           │
│   aspect = max(a,b) / min(a,b)     │
└────────────────────────────────────┘
```

### 3.1 Step 1 — 등가직경 De 계산

```
De = ROUND( ( (3.295 × 10⁻¹⁰ × Q^1.9 / R)^0.199 × 1000 ) / α , 0 )   [mm]
```

- 단위: Q [CMH], R [mmAq/m]
- 결과: De [mm] (정수 반올림)
- 출처: `D18` 수식

### 3.2 Step 2 — 보조 계산값

| 항목 | 공식 | 단위 |
|------|------|------|
| 단면적 A | `π · De² / 4 / 10⁶` | m² |
| 풍속 V | `1.2732 · Q / (3600 · De²) · 10⁶` | m/s |
| 동압 Δp | `0.061 · V²` | mmAq |
| 실제 저항 R′ | `0.00119 · V^1.9 · (De/1000)^(-1.22)` | mmAq/m |

> 풍속 공식의 1.2732 ≈ 4/π. `Q / (3600·A)`를 De² 형태로 정리한 결과.

### 3.3 Step 3 — 사각덕트 결정

#### Mode A : 단변 지정
- 입력: 단변 b [mm], 등가직경 De [mm]
- 출력: 장변 a [mm] (Huebscher 공식 기반, De 만족하는 최소 표준치수)
- 의사코드:
  ```
  for a in standardSizes (오름차순):
      if 1.3*(b*a)^0.625/(b+a)^0.25 >= De:
          return a
  ```

#### Mode B : 모든 조합 탐색 (단변 미지정)
- 입력: 등가직경 De [mm], 아스펙트비 한계 (기본 3.1)
- 출력: 조건을 만족하는 (b, a) 조합 리스트
- 알고리즘: §6.3 참조

### 3.4 Step 4 — 검증 (Huebscher 등가원형 공식)

```
De′ = ROUND( 1.3 × ( (b · a)^0.625  /  (b + a)^0.25 ) , 0 )   [mm]
```

- 의미: 결정된 사각덕트(b × a)의 등가원형 직경
- 기준: `De′ ≥ De` 가 되어야 풍량/저항 조건을 만족
- 아스펙트비: `max(a, b) / min(a, b) ≤ 3` (부득이 시 ≤ 5)

## 4. 환산표 구조와 LOOKUP 알고리즘

### 4.0 결론 — 환산표는 **수식 하나로 완전히 대체 가능**

엑셀 `환산표` 시트의 모든 데이터는 다음 Huebscher 공식의 결과를 미리 계산해 둔 것이다.

```
De_eq(b, a) = 1.3 · (b·a)^0.625 / (b + a)^0.25
```

따라서 프로그램에서는:
- **환산표 데이터 자체를 보관할 필요가 없다** (Huebscher 공식만 보유)
- **LOOKUP 보조 영역(5~7행)도 불필요** (표준치수 배열을 단순 순회로 대체)

아래 §4.1 ~ §4.4는 원본 엑셀의 동작 방식을 이해하기 위한 **참고 설명**이며, 본 프로그램에서는 구현하지 않는다.

### 4.1 데이터 영역 (8행 이후)

- F열: **단변 b** [cm] — 5, 10, 15, 20, … (10cm 간격)
- 7행: **장변 a** [cm] — 5, 10, 15, 20, … (10cm 간격)
- G8 이후 매트릭스: 각 (b, a) 쌍의 **등가원형 직경 [cm]**
  - 셀 수식: `=1.3*(($F·*·$7)^0.625/(F·+$7)^0.25)`

### 4.2 LOOKUP 보조 영역 (5~7행)

`F25`에서 사용하는 LOOKUP 범위는 `환산표!$A5:$IV7` 의 3행짜리 벡터다.

| 행 | 역할 | 내용 |
|----|------|------|
| 5행 | 검색 키 (구간 하한) | `G5=0`, `H5=G6+0.1`, `I5=H6+0.1`, … |
| 6행 | E25(단변)에 대한 각 장변별 De [cm] | `=VLOOKUP($F$1, 환산표2, 열번호, 0)` |
| 7행 | 결과 (장변) [cm] | 5, 10, 15, 20, … |

여기서 `F1 = E25/10` (단변을 cm로 환산).

### 4.3 LOOKUP 동작 원리

엑셀의 `LOOKUP(value, vector)`는 **첫 행에서 value 이하의 가장 큰 값**의 위치를 찾고, **결과 벡터의 같은 위치 값**을 반환한다. 첫 행은 오름차순이어야 한다.

핵심 아이디어:
- 6행 = “선택된 단변 b에서 장변을 (5, 10, 15, …) cm로 바꿀 때의 등가원형 직경”
  - 단변 고정 + 장변 증가 → 등가원형도 단조 증가
- 5행 = `이전 단계의 6행 값 + 0.1`
  - 즉, “장변 a₁일 때의 De보다 0.1cm라도 크면 다음 장변 a₂를 선택해야 한다”는 구간 분할

따라서 `LOOKUP(De_cm, 5행, 7행)`는

> “단변 b에서, 등가원형이 De_cm 이상이 되는 **최소 장변 a**”

를 반환한다.

### 4.4 예시 (단변 50cm 기준)

| 열 | G | H | I | J | … |
|----|---|---|---|---|---|
| 5행 (검색키) | 0 | G6+0.1 | H6+0.1 | I6+0.1 | … |
| 6행 (단변50일 때 De cm) | 22.6 | 30.9 | 37.7 | 43.4 | … |
| 7행 (장변 cm) | 5 | 10 | 15 | 20 | … |

(6행 값은 예시이며 실제는 환산표 수식 결과)

`De = 35 cm` 입력 시:
- 5행에서 35 이하의 최대 = `H5 = G6+0.1 ≈ 22.7` → H열 매칭
- 결과 = `H7 = 10 cm` … (실제로는 6행 값이 다르므로 매칭 위치도 달라짐)

> 단, ROUNDUP은 일부러 0.1 단위로 올려 “경계값에서 한 단계 위 장변을 선택”하도록 보장한다.
> `ROUNDUP(De/10, 2)` 이지만 De는 정수이므로 사실상 `De/10`을 그대로 cm로 변환하는 효과.

## 5. 핵심 공식 요약

```
[등가직경 — 풍량/저항 기반]
De = round( (3.295e-10 · Q^1.9 / R)^0.199 · 1000 / α )           [mm]

[등가원형 — Huebscher 공식 (사각 → 원형)]
De_eq(b, a) = 1.3 · (b·a)^0.625 / (b + a)^0.25                   [동일 단위]

[보조]
A    = π · De² / 4                                                [m² 환산 필요]
V    = Q / (3600 · A)                                             [m/s]
Δp   = 0.061 · V²                                                 [mmAq]
R′   = 0.00119 · V^1.9 · (De[m])^(-1.22)                          [mmAq/m]
```

## 6. 프로그램 모듈화 제안 (구조만)

### 6.1 클래스 / 모듈 구분

```
│   └── RectCombinationFinder  // Mode B: 모든 조합 탐색 (§6.3)

```

### 6.2 환산표 처리 방식

**채택: 공식만 사용 (Huebscher).** 환산표 데이터는 임베드하지 않는다. (§4.0 참조)

### 6.3 핵심 알고리즘 (의사코드)

#### Mode A — 단변 지정 시 최소 장변

```text
function FindMinLongSide(De: mm, b: mm, sizes: int[]) -> mm
    for a in sizes (오름차순):
        if a < b: continue                              // 장변은 단변 이상
        De_eq = 1.3 * (b*a)^0.625 / (b+a)^0.25
        if De_eq >= De:
            return a
    return -1   // 표준치수로 표현 불가
```

#### Mode B — 모든 조합 탐색 (아스펙트비 ≤ aspectMax)

```text
function FindAllRectCombinations(De: mm, aspectMax: double, sizes: int[]) -> RectCombination[]
    results = []
    for b in sizes (오름차순):
        for a in sizes where a >= b (오름차순):
            if (a / b) > aspectMax:
                break                                   // 이 b는 더 이상 유효한 a 없음
            De_eq = 1.3 * (b*a)^0.625 / (b+a)^0.25
            if De_eq >= De:
                results.add({
                    b, a,
                    De_eq,
                    aspect = a / b,
                    area   = b * a
                })
                break                                   // 해당 b의 최소 a 채택
    return results
```

**특성**:
- 결과 개수 = `aspectMax`를 만족하는 b의 개수 (b가 너무 크면 De 만족하는 a < b 가 되어 제외됨)
- b 오름차순 정렬 (필요 시 면적·아스펙트비로 재정렬 가능)
- 각 b당 최소 a만 1개 → 과대설계 자동 배제
