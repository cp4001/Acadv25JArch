> 근거 도서: 도서출판 한미 「덕트 설계시공편」, 「건축설비설계메뉴얼 - 공기조화설비설계(지문당)」

## 1. 개요

본 설계서는 `DUCT_MEASURE_1204.xls`에 구현된 덕트 사이즈 결정 로직을 분석·정리한 문서이다. 엑셀 시트에는 5가지 입력 시나리오가 존재하며, 각 시나리오는 **(Q, D, V, R, △p)** 5개 변수 중 일부를 입력받아 나머지를 계산하고, 최종적으로 **원형덕트 직경 → 사각덕트(장변×단변)** 로 변환한다.

### 1.1 핵심 변수

| 기호 | 명칭 | 단위 | 비고 |
|------|------|------|------|
| Q | 풍량 (Air Flow) | CMH (m³/h) | 입력값. 입력 단위 환산 필요 |
| D, De | 원형 직경 / 등가 원형 직경 | mm | 사각덕트의 마찰손실 등가 직경 |
| V | 풍속 (Velocity) | m/s | |
| R | 단위 마찰손실 (정압 강하) | mmAq/m | 통상 `정압`으로 호칭 |
| △p | 동압 (Velocity Pressure) | mmAq | △p = ρV²/2g, ρ_air ≈ 1.2 |
| A | 단면적 | m² | A = πD²/4 또는 a×b |
| a, b | 사각덕트 장변, 단변 | mm | a ≥ b |
| 여유율 | 단위면적 보정계수 | - | 1.00 ~ 1.10 (실제 저항 확인 필요) |
| ASP | 종횡비 (Aspect Ratio) | - | a/b. 권장 ≤ 3 |

### 1.2 기본 공식

```
Q = A × V                       … 연속 방정식 (Q는 m³/s 환산 필요)
A = π × D² / 4                  … 원형 단면적
A = Q / V                       … 풍속·풍량으로 면적 산출
△p = ρ × V² / 2g                … 동압 (mmAq)  [ρ=1.2 kg/m³, g=9.81]
De = 1.30 × (a·b)^0.625 / (a+b)^0.25   … 등가원형 직경 (Huebscher)
```

마찰손실 R은 Darcy-Weisbach 식 또는 경험식(예: ASHRAE 마찰선도)으로 (Q, D)로부터 산출되며, 엑셀 내부에서는 `환산표` / `정방형환산표` 시트의 보간 테이블로 구현되어 있다.

## 2. 시트 구조 (DUCT MEASURE)

| 영역 | 셀 | 내용 |
|------|----|------|
| 입력 | E6 | Q (풍량) [CMH] |
| 입력 | E7 | D (원형 직경) [mm] |
| 입력 | E8 | R (정압, 단위 마찰손실) [mmAq/m] |
| 입력 | C8 | 여유율 |
| 입력 | E9 | V (풍속) [m/s] |
| 결과 (1) Q+R → D | E12~E17 | Q, A, V, △p, R, **De** |
| 결과 (2) D+V → Q | G12~G17 | Q, A, V, △p, R, NECK DIA |
| 결과 (3) Q+D 검토 | I12~I17 | Q, A, V, △p, R, NECK DIA |
| 사각 변환 (1) | E24~E25 | 입력 단변 → 장변 (정압 등가) |
| 사각 변환 (3) | G24~G25 | 입력 단변 → 장변 (등가 원형) |
| 아스펙트비 결정 | F32~F33 | ASP 입력 → 장변·단변 자동 결정 |
| 치수 변환 | I2~K3 | 기존 (a₀, b₀) + 변경치수 → 상대치수 (등가 원형 유지) |

내부 참조 시트:
- `DUCT_DATA` — 권장 풍속표(주택/사무소/공장), 단위 마찰손실 추장치
- `환산표`, `환산표 (2)` — (단변, 장변) → R(또는 De) 2D 룩업 테이블 + 표준 사이즈 시리즈
- `정방형환산표` — 원형 직경 De → 표준 사각 장변 매핑 테이블
- `배관관경결정`, `관경선정표`, `손실수두에따른유량및관경` — 배관용 부속 테이블

## 3. 5가지 결정 시나리오

### 시나리오 1: 풍량 + 정압 → 덕트 사이즈

가장 일반적인 설계 흐름. 풍량(Q)과 허용 단위 마찰손실(R)이 주어지면 원형 직경 → 사각으로 변환한다.

```
입력: Q [CMH], R [mmAq/m]                ← E6, E8
       (팬 토출측 R ≈ 0.10 mmAq/m,  흡입측 R ≈ 0.08 mmAq/m 기준)
       여유율 (선택, 1.00~1.10)            ← C8

처리:
  1) (Q, R) → D   (마찰선도/환산표 보간)
  2) A = πD²/4
  3) V = Q / A
  4) △p = ρV²/2g

출력: D, A, V, △p, R, **De**            ← E12~E17

사각 변환:
  - 사용자가 단변(또는 장변) b 입력 (E25)
  - 같은 정압을 유지하는 장변 a 산출 (F25)
  - 같은 등가원형(De)을 유지하는 a 별도 산출
  → 두 결과 차이가 발생하면 “면적은 커지지만 정압은 유지” 의미

검증:
  - ASP = a/b 가 표시됨 (E30)
  - **ASP ≤ 3** 인지 확인. 초과 시 사각비 재조정
```

장변이 길어지고 단변이 짧아질수록 마찰손실은 증가하므로, **같은 정압·풍량을 유지하려면 더 큰 면적이 필요**하다는 점이 설명서 line 9에 명시되어 있다.

### 시나리오 2: 원형직경 + 풍속 → 풍량

기존 원형덕트의 풍속 조건이 정해진 경우 통과 가능한 풍량을 역산한다.

```
입력: D [mm], V [m/s]                     ← E7, E9

처리:
  1) A = πD²/4
  2) Q = A × V × 3600          (m³/s → CMH)
  3) △p = ρV²/2g
  4) R = (Q, D) 룩업

출력: Q, A, V, △p, R, NECK DIA          ← G12~G17
```

### 시나리오 3: 풍량 + 직경 → 검토 (Q + D 검토)

설계 중간 단계 검토용. 풍량과 임의의 직경을 넣어 면적·풍속·동압·정압이 적정 범위인지 확인한다.

```
입력: Q [CMH], D [mm]                     ← E6, E7

처리:
  1) A = πD²/4
  2) V = Q / A
  3) △p = ρV²/2g
  4) R = (Q, D) 룩업
  5) NECK DIA = Q를 15 m/s로 통과시킬 때 직경 (디퓨저 NECK 산정)

출력: Q, A, V, △p, R, NECK DIA          ← I12~I17

용도:
  - 권장 풍속(주덕트 6~10, 지덕트 5~6 m/s 등) 위반 여부 확인
  - 단위 마찰손실 추장치(0.07~0.15 mmAq/m) 위반 여부 확인
```

### 시나리오 4: 풍량 + 풍속 → 덕트 사이즈

풍속을 직접 지정하는 경우(예: 디퓨저 NECK, 흡입그릴).

```
입력: Q [CMH], V [m/s]                    ← E6, E9

처리:
  1) A = Q / V
  2) D = √(4A/π)             (시나리오 3의 J18 = 원형 직경)
  3) (D, ASP 또는 장변) → 단변 결정

출력: 사각 (a, b)
```

### 시나리오 5: 기존 사각덕트 → 새 사각덕트 (등가 변환)

치수 제약(천장고, 슬리브 등)으로 기존 덕트를 다른 형상으로 바꿀 때 사용. **등가원형(De) 보존**이 원칙.

```
입력: 기존 (a₀, b₀)                        ← I2, J2
       변경할 한 변 a₁ (또는 b₁)           ← I3 또는 J3

처리:
  1) De₀ = 1.30 × (a₀·b₀)^0.625 / (a₀+b₀)^0.25
  2) De₀ = 1.30 × (a₁·b₁)^0.625 / (a₁+b₁)^0.25  를 만족하는 b₁ 역산
  3) ASP = a₁/b₁ (또는 b₁/a₁) 표시

출력: 변경치수 (a₁, b₁), 등가원형 De₀
```

### 시나리오 부가: 여유율(Safety Margin) 적용

```
A_design = A_calc × 여유율            (여유율 < 1.0 일 때 사이즈 축소)
```

여유율 < 1.0 을 적용하면 덕트 사이즈가 줄어들지만 **실제 저항(R, △p) 값이 허용치를 초과할 수 있으므로 반드시 검토 시나리오로 재확인**해야 한다. (설명서 line 34 명시)

## 4. 종합 결정 플로우

```
┌──────────────────────────────────────────────┐
│  입력 변수 조합 식별                          │
│   - Q + R   →  시나리오 1 (가장 일반적)       │
│   - D + V   →  시나리오 2                    │
│   - Q + D   →  시나리오 3 (검토)             │
│   - Q + V   →  시나리오 4                    │
│   - 기존 덕트 (a₀,b₀) + 변경변 → 시나리오 5  │
└──────────────────────────────────────────────┘
                    │
                    ▼
┌──────────────────────────────────────────────┐
│  원형 등가 직경 De 산출                       │
│   - 시나리오별 공식·테이블 보간              │
│   - 여유율 적용 (선택)                        │
└──────────────────────────────────────────────┘
                    │
                    ▼
┌──────────────────────────────────────────────┐
│  사각덕트 (a × b) 변환                        │
│   방법 A: 단변(또는 장변) 직접 입력 →        │
│            상대 변을 등가원형/정압 보존으로   │
│            역산                              │
│   방법 B: ASP 직접 입력 → 두 변 동시 결정     │
└──────────────────────────────────────────────┘
                    │
                    ▼
┌──────────────────────────────────────────────┐
│  검증 (Validation)                            │
│   1) ASP = a/b ≤ 3                           │
│   2) V ∈ 권장 풍속 범위 (DUCT_DATA)          │
│   3) R ∈ 단위 마찰손실 추장치                │
│   4) 여유율 적용 시 R, △p 재확인             │
└──────────────────────────────────────────────┘
                    │
                    ▼
              최종 (a, b) 확정
```

### 5.4 원형 De → 사각 장변 매핑 (정방형환산표)

### 5.5 환산표 — (장변, 단변) → 등가원형 De 룩업 매트릭스

`환산표` 시트의 본체는 **69 × 69 = 4,761 셀**의 2D 룩업 매트릭스이다. 행은 장변, 열은 단변, 셀 값은 등가원형 직경 De 이다.

**구조**

```
        단변 [cm] →  5   10   15   20   25  ...  490  500
장변 [cm]
   5   →           5.47 7.62 9.13 10.3 11.4 ...  36.5  ...
  10   →           7.62 10.9 13.3 15.2 16.9 ...  56.1  ...
   :
 500   →             :    :    :    :    :        :   546.6
```

- **단위**: 행/열 헤더는 cm, 셀 값(De)도 cm. 결과 표시 시 ×10 하여 mm.
- **대칭성**: De(a, b) = De(b, a). 즉 매트릭스는 주대각 기준 대칭. 단변·장변 구분이 없는 형태로 사용 가능.
- **공식 일치**: 셀 값은 Huebscher 식 `De = 1.30 × (a·b)^0.625 / (a+b)^0.25` 와 정확히 일치 (검증: a=10, b=5 → 표값 7.62 vs. 계산값 7.62).
- **사용처**:
  - 사용자가 (장변 a, 단변 b)를 입력 → De를 즉시 룩업 → 마찰선도 또는 시나리오 1·3 결과 산출
  - 역방향: De가 주어진 상태에서 장변 a 고정 시 → 같은 행에서 De에 가장 가까운 셀의 단변 b를 역산

### 5.6 종횡비 (Aspect Ratio)

> "직사각 닥트의 형상은 되도록 정사각형에 가까운 것으로" — 시트 하단 주석

- 권장: ASP ≤ 2
- 한계: **ASP ≤ 3** (이상 시 마찰손실 급증, 시공성 저하)
- 시트의 ASP 입력 박스는 1 ~ 8 까지 선택 가능하지만, 8 입력 시 경고 의미로 표기됨

## 6. 구현 시 유의점 (CAD/프로그램 이식 관점)

1. **단위 일관성**
   - Q 입력은 CMH(m³/h). 내부 계산은 m³/s (÷3600) 변환 후 수행.
   - 면적은 m², 직경은 mm — 공식 적용 시 단위 환산 누락 주의.

2. **마찰선도 룩업 vs. 공식**
   - 엑셀은 `환산표`·`정방형환산표`의 2D 보간으로 (Q, R) → D 를 구함.
   - 프로그램화 시에는 다음 둘 중 선택:
     - (a) 시트의 룩업 테이블을 그대로 이식하여 보간
     - (b) ASHRAE 마찰선도 공식 (Darcy-Weisbach + Colebrook-White) 직접 계산
   - 일관된 결과가 필요하면 (a) 권장.

3. **장변·단변 스냅(snap)**
   - 계산된 임의 치수를 그대로 출력하지 말고, **§5.3 표준 사이즈 시리즈**의 가장 가까운 값(또는 올림값)으로 스냅 처리.
   - 정방형환산표의 De → 장변 룩업처럼 "구간 매핑" 방식으로 결정성 확보.
   - 스냅 후에는 R, V, △p 를 재계산하여 추장치 위반 여부 재확인.

4. **여유율의 의미**
   - 단순히 면적에 곱하는 보정값. 음수·0 입력 방지.
   - 1.0 미만이면 사이즈 축소 → 반드시 R/△p 재계산 후 추장치 비교.

5. **ASP 권고 위반 처리**
   - 자동 계산 결과 ASP > 3 이면 경고 표시. 사용자가 단변을 늘려 재계산하도록 유도.

6. **사각 변환의 두 갈래**
   - "정압 보존" 변환과 "등가원형 보존" 변환은 **결과 면적이 다를 수 있다.**
   - 시나리오 1은 정압 기준, 시나리오 3·5는 등가원형 De 기준.
   - 어느 기준으로 변환했는지 결과 표시에 명시할 것.

7. **NECK DIA**
   - 디퓨저/그릴의 목 직경(neck) 산정용. V=15 m/s 기준 직경.
   - 분기 말단 부속(diffuser/grille) 선정 시 사용되며, 본선 덕트 사이즈와는 별개 출력.
